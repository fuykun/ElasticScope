Choose Your Installation Method

Run from Source

Best for development and modification. Requires Node.js.

# Clone repository git clone https://github.com/fuykun/ElasticScope.git cd ElasticScope # Install dependencies npm install # Start frontend and backend concurrently npm run dev:all

Pull from Docker Hub

Quickest way to get started without installing Node.js.

# Run latest image docker run -d -p 3000:3000 -p 5000:5000 \ ghcr.io/fuykun/elasticscope:latest

Build Locally

Build the container image on your own machine.

# Clone & Enter git clone https://github.com/fuykun/ElasticScope.git cd ElasticScope # Build Image docker build -t elasticscope . # Run Container docker run -d -p 3000:3000 -p 5000:5000 elasticscope
